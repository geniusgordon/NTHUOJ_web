##Install Guide
---

###Get our project
```
git clone git@github.com:bruce3557/NTHUOJ_web.git
```

####Dependencies: 
* python2.7 
* django1.7
* mysql-server
* python-mysqldb

###Installation:
* Execute install.py for initial setting.
    ```
    python install.py
    ```

* After installing, you can modify `'<project/root>/nthuoj.ini'` if your database setting is modified.
